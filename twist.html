Background

A graph or network for our purposes is a set of relationships between things. Two vertices are adjacent if there is an edge between them. We can concisely represent a given graph between a fixed number of vertices using an adjaceny matrix. An adjacency matrix for a graph is a symmetric square matrix, where the index for rows or the columns is some ordering of the vertices and we put a one for location and if vertices are connected by an edge.
